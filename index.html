Favorites API

getAllUsers()[GET]

Returns embedded json of all users.

query format:
https://favorites-api.herokuapp.com/getAllUsers

user_favorite_genre(user_id, genre)[GET|POST|PUT]

Returns json of favorite books from
spcific user id by ganre.

query format:
https://favorites-api.herokuapp.com/user_favorite_genre/user_id/genre

getFavoritesById(user_id)[POST]

Returns json of user by id.

query format:
https://favorites-api.herokuapp.com/getFavoritesById/
